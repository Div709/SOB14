# SOB14
Code for SOB 14
This codes make the servo move according to the distance of obstacle from the receiver of the ultrasonic sensor.
A RGB also turns on according to the distance.
Similarly, a buzzer will sound off when an obstacle is too close.
Part of my thinking was to maybe use it in a helmet used by visually impared indivuduals.
When an obstacle is too close, the buzzer will sound off and alert the individual
